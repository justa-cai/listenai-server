feat: 实现 ListenAI 语音助手云代理服务

新增完整的 cloud 模块，实现 WebSocket 服务器作为语音助手的云代理层：

- server.py: WebSocket 服务器主入口，处理客户端连接和消息路由
- session.py: 客户端会话管理，维护连接状态和消息处理流程
- protocol.py: 定义客户端与服务器间的消息协议结构
- asr_client.py: ASR 语音识别服务的 WebSocket 客户端
- tts_client.py: TTS 语音合成服务的 WebSocket 客户端
- llm_client.py: LLM 大语言模型的 HTTP 客户端
- mcp_manager.py: MCP (Model Context Protocol) 工具调用集成
- audio_buffer.py: 音频数据缓冲区管理
- config.py: 基于 environment 的配置管理
- test_client.html: 完整的 HTML 测试客户端
- requirements.txt: 项目依赖 (websockets, aiohttp, python-dotenv)
